# afisha_service_analyse
Анализ данных Яндекс Афиша для  одного из итоговых модулей по обучению

###
В папке `data` лежит файл с данными о курсе валют между тенге и рублем 

###
В папке `src` лежит основной файл с аналищом и исследованием `afisha_analyse.ipynb`

####
В файле `requirements.txt` сохранены данные о библиотеках, используемых для выполнения работы.
